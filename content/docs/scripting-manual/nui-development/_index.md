**NUI** （“新用户界面”的缩写）是CitizenFX框架中基于HTML的用户界面功能。 目前正在使用
[Chromium Embedded Framework](cef)，它提供了一种异步，高性能的方式来创建游戏界面，方法是使用
网络技术（HTML / CSS / JS，包括诸如React或Angular之类的框架，以及加速的WebGL）。
